# 📝 To-Do List App

A clean and responsive **To-Do List Application** built using **HTML, CSS, and JavaScript**. This project allows users to add and delete tasks dynamically using DOM manipulation and event delegation.

## 💡 How It Works

- Users enter a task in the input field.
- Clicking **Add Task** creates a new list item dynamically.
- Every task includes a **Delete** button.
- Clicking **Delete** removes the corresponding task using **Event Delegation**.
